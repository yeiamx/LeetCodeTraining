解题思路：
首先我们S3的长度必须是i+j，我们在这个情况下定义状态
状态:dp[i][j] = S1[0..i-1] S2[0..j-1]是否能够拼出S3[0..i+j-1] (Bool)

转移：分析所有可能的拼法，有一种拼法为true dp[i][j] = true
所有的拼法里 我们根据S3[i+j-1]的来源进行分类 它要么来自S1[i-1] 要么来自S2[j-1]
if S3[i+j-1] != S1[i-1] && S3[i+j-1] != S1[j-1] false
if (S3[i+j-1] == S1[i-1] && dp[i-1][j]) true
if (S3[i+j-1] == S1[j-1] && dp[i][j-1]) true

边界：dp[0][0] = true
dp[0][0-m]: S2与S3的匹配结果
dp[0-n][0]: S1与S2的匹配结果
